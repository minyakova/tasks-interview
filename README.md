## Запуск программы:
1. Запустите docker compose в корне проекта:
```
docker-compose up
```
2. В контейнере php запустите composer(для папки vendor и установки зависимостей):
```
composer install
```
3. Запустите сайт локально в браузере:
```
http://mysite.local/
```
